[Multipole Frames](#multipole-frames.png)

### Multipole Parameters
```
M1 M2 M3      q
              Dx  Dy  Dz
              Qxx
              Qxy  Qyy
              Qxz  Qyz  Qzz
```

### Z-then-X
* M1 M2 M3 

### bisector
* M1 M2 -M2, or M1 -M2 M2, or M1 -M2 -M2

### bisector-then-X
* M1 M2 -M3 -M3

* typically Dy, Qxy, Qyz components are zero, unless it is chiral such as C_alpha in protein
